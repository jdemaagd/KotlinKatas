# Sorting Algorithms

## Big O

|         Algorithm         |     Performance      |
|:-------------------------:|:--------------------:|
|        Bubble Sort        |        O(n^2)        |
|      Selection Sort       |        O(n^2)        |
|      Insertion Sort       |        O(n^2)        |
|        Merge Sort         |      O(n log n)      |
|        Quick Sort         | O(n log n) to O(n^2) |
|         Heap Sort         |        O(n^2)        |
